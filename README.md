### 项目目前已有功能

1.登录


2.菜单展示(支出类型、支出详情、统计分析等)


3.支出类型列表展示、添加支出类型、删除支出类型;


4.支出详情列表展示、添加支出详情、删除支出详情;


5.统计分析(今日总支出、昨日总支出、最近7天总支出、最近15天总支出、本月总支出、上个月总支出)
